  ##### Python的运算

  运算符：`+、-、*、/、//、%、**`
  ##### python做简单的加减乘除在解释器里面直输入即可

  ```angular2
  >>> 1+1  # 加 1+1输出2
  2 
  >>> 5-2  # 减 5-2输出3 6-8输出-2
  3
  >>> 2*3  # 乘 2*3输出6
  6
  >>> 10/2 # 除 10/2输出5
  5.0
  >>>10//3 # 取整 10//3输出3或10.0//3.0输出3.0
  3
  >>>6 % 4 # 取余 10/2输出5
  2
  >>>2**3  # 幂 2**3输出8
  8
  ```
